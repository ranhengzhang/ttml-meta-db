# ttml-meta-db

> 一个用来管理 TTML 元数据的软件，可以一键复制歌曲所有元数据。

## 使用

前往 [Release](../../releases) 页面或 [Actions](../../releases) 页面下载最新发布版本，直接运行。

## 反馈

在 [Issue](../../issues) 页面发起提交，提交前请查看是否有重复问题。
